# oracle-forge-data-agent
TRP 1 Week 8-9

## Web Query Console

Run a black-and-white web UI for querying Postgres, SQLite, MongoDB (via Toolbox) and DuckDB (local file):

```bash
cd /week8-9/oracle-forge-data-agent
python3 -m pip install -r requirements.txt
python3 app.py
```

Open `http://127.0.0.1:8080`.
